Installation instructions

(This will potentially be even more streamlined in the future)

  1. Download ATLauncher (The .jar file is a portable version)
  2. Once installed, log in with your minecraft account.
  3. Press the "Instances" button, and select "import" in the top left of the window
  4. Give it this .zip folder: Download
  5. Once it's finished setting up, start the instance.
  6. Add a server with ip: mc.dovas.se
  7. Have fun!

Mod briefing and documentation links

Large, content-rich mods

  1. CC: Tweaked (Programmable computers)

    • Fully integrated custom OS shell
    • LUA Scripting Language (Python-like syntax)
    • Sensors and detectors can automate behaviour
    • Robots can mine, sort inventories, harvest crops, etc.
    • (How I learned programming)

  2. Thermal Series (Expansion, Dynamics, Cultivation) (Tech/industrial mod)

    • Process materials futher than before
    • Create complex machines that can automate chores
    • Craft powered armor and weapons

  3. Botania (Magical botany mod)

    • Create magic items and tools for improved movement, efficiency, building, etc.
    • Everything you need to know, along with a progression system, can be found in the in-game book Lexica Botanica that you should spawn with. It is otherwise easy to craft.

Smaller, quality of life mods

  1. CarryOn (Pick up chests without breaking them)

    • Just bind a key (shift by default) to hold while right clicking a chest to pick it up. You move slower and cannot use tools while carrying a chest.

  2. JustEnoughItems (Search for, and see item recipes in your inventory)

  3. JourneyMap (Minimap in corner of screen, set waypoints)

    • Doesn't really need the documentation, just use the in-game map menu by pressing J (by default).
    • Bind hotkey controls in the Controls tab in-game.

  4. Waystones (Teleportation beacons that use XP, allows distant places to be connected)

    • Waystones can also be found already placed in most villages.

  5. Hats (Find and wear cool hats as uncommon drops from mobs)

  6. Iron Chests (Reinforce chests with metals for larger storage)

  7. Openblocks Elevators (Simple elevators for vertical movement)
